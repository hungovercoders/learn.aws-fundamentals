# Sources and inspiration — learn.aws-fundamentals

hungovercoders is dataGriff's path through the AWS Certified Cloud Practitioner (CLF-C02) syllabus. The canonical references below are where AWS itself defines the material; this series is a guided walk alongside them, not a replacement.

## Canonical reference

- [AWS Certified Cloud Practitioner — official exam page](https://aws.amazon.com/certification/certified-cloud-practitioner/) — registration, exam guide (CLF-C02 PDF), sample questions, and the official skill set
- [AWS Well-Architected Framework documentation](https://docs.aws.amazon.com/wellarchitected/latest/framework/welcome.html) — the canonical six-pillars reference
- [AWS Documentation](https://docs.aws.amazon.com/) — the umbrella site for every service the exam covers (per-service FAQ pages are linked from each lesson where relevant)
- [AWS Global Infrastructure](https://aws.amazon.com/about-aws/global-infrastructure/) — current region / AZ / edge location list, always up to date
- [AWS Shared Responsibility Model](https://aws.amazon.com/compliance/shared-responsibility-model/) — the official customer-vs-AWS split

## Recommended alternative paths

- [Stéphane Maarek — AWS Certified Cloud Practitioner course (Udemy)](https://www.udemy.com/course/aws-certified-cloud-practitioner-new/) — the most popular video-led alternative; reach for it if you learn better watching than reading
- [Adrian Cantrill — AWS Certified Cloud Practitioner course](https://learn.cantrill.io/p/aws-certified-cloud-practitioner) — deeper-than-Foundational hands-on labs; reach for it if you intend to continue to SAA-C03 and want the lab grounding early
- [AWS Skill Builder — Cloud Practitioner Essentials](https://explore.skillbuilder.aws/learn/learning_plan/view/82/plan) — AWS's own free learning path; reach for it for vendor-canonical pacing

## Practice and reinforcement

- [Tutorials Dojo — AWS Certified Cloud Practitioner Practice Exams (CLF-C02)](https://portal.tutorialsdojo.com/courses/aws-certified-cloud-practitioner-practice-exams-clf-c02/) — the single biggest pass-rate lever after this series. Six full-length practice exams with explanations.
- [AWS Skill Builder — Cloud Quest: Cloud Practitioner](https://explore.skillbuilder.aws/learn/course/external/view/elearning/11458/aws-cloud-quest-cloud-practitioner) — free gamified walkthrough; useful in parallel with the compute / storage / database lessons
- [AWS Skill Builder — official CLF-C02 sample questions (free)](https://explore.skillbuilder.aws/learn/course/12483/aws-certified-cloud-practitioner-official-question-set-clf-c02-english) — 20 official sample questions in AWS's house style

## Communities

- [AWS re:Post](https://repost.aws/) — AWS's own community Q&A, supersedes the old AWS Forums
- [r/AWSCertifications on Reddit](https://www.reddit.com/r/AWSCertifications/) — pass-rate stories, study tips, fresh exam impressions

## How to use this series

Read this series for the opinionated walk through the CLF-C02 syllabus. Open the canonical references when you need authoritative detail (e.g. the *exact* list of EC2 instance types or current Region count). Take an alternative path if dataGriff's voice isn't landing for you — Maarek's videos, Cantrill's labs, and AWS Skill Builder all cover the same syllabus differently. The canonical source is the same either way.

When the series is done, take at least three Tutorials Dojo practice exams. Booking the real exam first is the most reliable way to make the rest happen — see lesson 01.
