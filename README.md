# learn.aws-fundamentals

A hands-on tutorial that takes a technically literate reader from "I have an AWS account" to **passing the AWS Certified Cloud Practitioner exam (CLF-C02)**. Covers all four exam domains in roughly the weighting the exam uses, with hands-on labs where they help retention and concept lessons where the exam tests recognition. Every lesson ends with five CLF-C02-style sample questions to build exam-pattern recognition alongside the AWS knowledge itself.

Read the full tutorial: **[hungovercoders.com/training/aws-fundamentals](https://hungovercoders.com/training/aws-fundamentals)**
